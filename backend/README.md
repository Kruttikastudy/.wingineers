# Wingineers — Cyber Defense Backend API

FastAPI backend providing deepfake detection, phishing analysis, voice interception, prompt injection detection, and XAI explainability services.

## Features
- **Deepfake Detection** — Audio, video, and image analysis with ML models
- **Phishing Analysis** — URL and email threat intelligence
- **Voice Intercept** — Real-time Twilio webhook processing
- **Prompt Injection Detection** — LLM prompt guard
- **XAI Dashboard** — SHAP-based explainability
- **Google Auth + MongoDB** — User management
